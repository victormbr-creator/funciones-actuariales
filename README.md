# Funciones Actuariales

Librería en Python para cálculos actuariales discretos escritos en español.

## Contenido

Actualmente incluye funciones para:

- Anualidades vitalicias
- Anualidades temporales
- Anualidades diferidas
- Seguros vitalicios
- Seguros temporales
- Seguros diferidos
- Seguros dotales
- Seguros crecientes
- Seguros decrecientes
- Funciones de conmutación
- Valor presente de flujos
